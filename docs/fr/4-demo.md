# Démo publique

La démo est maintenant disponible sur [demo.pwnthemall.com](https://demo.pwnthemall.com)

La base de données est restaurée chaque nuit. Par défaut, deux comptes sont créés.  
Vous pouvez essayer et expérimenter les fonctionnalités avec ces comptes :

| Nom d'utilisateur | Mot de passe |
|:----------------:|:------------:|
| user             | user         |
| admin            | admin        |

Veuillez noter que certaines fonctionnalités sont restreintes dans cette démo publique pour des raisons de sécurité.  
Si vous souhaitez découvrir toutes les fonctionnalités de Pwnthemall, veuillez consulter les pages d'installation et d'utilisation.
